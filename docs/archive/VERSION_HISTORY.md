# Histórico de Versões - Sport Club de Natal

Este documento registra a evolução do sistema de gestão e portal do sócio, seguindo o padrão SemVer adaptado:
- **0.0.x**: Correções, pequenas implementações e ajustes finos.
- **0.x.0**: Novas funcionalidades principais ou repaginação visual.
- **x.0.0**: Mudança completa na estrutura da aplicação.

---

## [0.3.2] - 2024-05-20
### Adicionado
- **Portal do Voluntariado**: Novo módulo acessível a todos os perfis para gestão de ações sociais e apoio ao clube.
- **Login Integrado**: Acesso unificado via dropdown no menu superior (Modo Visitante por padrão).
- **Proposta de Sócio**: Fluxo de pré-cadastro para interessados.
- **Gestão de Tarefas (Ex-Kanban)**: Renomeado e otimizado para gestão de atividades internas.
- **Histórico de Versões**: Documentação detalhada da evolução do app.

## [0.3.1] - 2024-05-18
### Adicionado
- **Dashboard do Treinador Revitalizado**: Estatísticas de performance e agenda real.
- **Ficha do Associado**: Visualização detalhada de dados financeiros e filiação.
- **Sistema de Presença**: Gestão de frequência em treinos via Server Actions.

## [0.3.0] - 2024-05-15
### Repaginação
- **Identidade Visual 2.0**: Implementação rigorosa das cores oficiais (Vermelho Sangue, Preto, Ouro) e design Glassmorphism.
- **Módulo de Gamificação**: Implementação de níveis, pontos e conquistas vinculadas à frequência.
- **Carteirinha Digital**: Geração dinâmica de QR Code para acesso ao clube.

## [0.2.14] - 2024-05-10
### Adicionado
- Autenticação básica via NextAuth.
- Integração inicial com Prisma e SQLite para desenvolvimento.
- Dashboard dinâmico para Sócios.

## [0.2.10] - 2024-05-02
### Adicionado
- Loja oficial do clube (Store) com gestão de estoque básica.
- Galeria de imagens com upload e listagem dinâmica.

## [0.2.5] - 2024-04-25
### Adicionado
- Módulo de Reserva de Barcos com verificação de disponibilidade.
- Painel de Notícias e comunicados oficiais.

## [0.2.0] - 2024-04-15
### Repaginação
- Transição para Next.js 14 App Router.
- Refatoração completa da arquitetura visual para Mobile-First.

## [0.1.12] - 2024-04-05
### Adicionado
- Estrutura base de permissões (Diretoria, Sócio, Treinador).
- Componentes UI fundamentais (Button, Card, Input).

## [0.1.5] - 2024-03-25
### Adicionado
- Protótipo inicial da Landing Page.
- Configuração do ambiente de desenvolvimento no Windows.

## [0.1.1] - 2024-03-10
### Inicial
- Criação do repositório e estrutura base do projeto.
- Definição do design system inicial.
